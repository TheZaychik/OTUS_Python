### Log Analyzer
Выполнено на версии Python 3.12

Запуск скрипта:
```shell
$ python3 log_analyzer.py
```
Опция --config имеет стандартный путь ./config.json

Запуск теста:
```shell
$ python3 -m unittest test.py
```
Для работы необходим файл report-template.html, так как он используется как шаблон
